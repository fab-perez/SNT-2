    HTML

    Depuis la naissance du Web en 1989, le langage HTML (HyperText Markup Language) est un des trois piliers du World Wide Web, avec le protocole Hypertext Transfer Protocol (HTTP) et les adresses Web (URL).     Balises

    HTML n'est pas un langage de programmation, c'est un langage de balisage conçu pour représenter une pages web et son contenu.

    Prenons un exemple de code écrit en HTML.

    <p>Mon premier paragraphe en HTML</p>

    On appelle cette ligne de code un élément HTML. Il comporte trois parties :

    • <p> est la balise ouvrante de l'élément ;
    • «Mon premier paragraphe en HTML» est le contenu de l'élément ;
    • </p> est la balise fermante correspondante (noter le « / ») ;

    19. Attributs

    On peut modifier encore plus l'apparence d'une page web, par exemple pour afficher un paragraphe en bleu, on utilise l'attribut style="color:blue;" à l'intérieur de la balise ouvrante <p> :

    <p style="color:blue;">Je suis écrit en bleu</p>

    Les éléments peuvent contenir des informations supplémentaires pour modifier la façon dont ils s'affichent : ce sont des attributs placés à l'intérieur d'un balise.

    Il existe de très nombreux attributs de style pour mettre en forme l'apparence d'une page web.

    Liens hypertexte

    Les liens hypertexte qui relient les pages web entre elles par un simple clic sont un aspect fondamental du web.

    L'élément <a> (pour ancre ou anchor en anglais) permet de créer un lien hypertexte. La balise ouvrante <a> doit comporter un attribut href (pour « hypertexte reference»), indiquant la destination du lien.

    Par exemple pour afficher le texte « Wiki » avec un lien vers le site https://www.fr.wikipedia.org on utilise l'attribut href="https://www.fr.wikipedia.org" à l'intérieur de la balise ouvrante <a> :

    <a href="https://www.fr.wikipedia.org">Wiki</a>

    25. Images

    L'élément <img> (pour image) permet d'afficher une image. La balise <img> est une balise orpheline (pas de balise fermante). Elle doit comporter deux attributs :

    • un attribut src (pour source),  indiquant le fichier de l'image.
    • un attribut alt (pour alternatif),  indiquant un texte alternatif si l'image ne s'affiche pas.

    Par exemple pour afficher une image « dauphin.jpg » située dans le répertoire image, on utilise les deux attributs src="images/dauphin.jpg" et alt="Un dauphin dans la mer" à l'intérieur de la balise ouvrante <img> :

    <img src="images/dauphin.jpg" alt="Un dauphin dans la mer">

    Que fait ce bouton ?

    On va maintenant combiner les deux approches en utilisant une fonction JavaScript. Une fonction est un bloc d'instructions qui permet d'effectuer une tache particulière quand elle est appelée par un événement qui se produit, par exemple si un bouton est cliqué.

    Attention que le corps d'une fonction JavaScript est délimité par des accolades {â€¦ } et toutes ses instructions se terminent par un point-virgule.

    Au lieu d'ouvrir une fenêtre pour afficher le message, on va maintenant afficher le message directement sur la page. Pour cela, on utilise la fonction document.getElementById(id) qui permet d'accéder à un élément HTML précis de la page. L'attribut id définit cet élément HTML, et la propriété innerHTML définit son contenu.

    }
    </script>
