    Cascading Style Sheets (CSS)

    CSS (pour Cascading Style Sheets, ou feuilles de style en cascade en français) est le code utilisé pour mettre en forme des pages web.

    De la même façon que HTML, CSS n'est pas un langage de programmation. C'est un langage de feuille de style, c'est-à-dire qu'il permet de mettre en forme un document HTML.

    On a vu dans l'activité précédente comment modifier l'apparence d'un élément HTML en utilisant un attribut à l'intérieur d'une balise. Par exemple, pour afficher un paragraphe en bleu foncé, on utilise l'attribut style="color:darkblue;" à l'intérieur de la balise ouvrante <p> :

    <p style=color:darkblue;"">Je suis écrit en bleu foncé</p>

    Mais ce n'est pas recommandé ! La bonne façon d'écrire une page web est de séparer le contenu de la page (le code HTML) de sa forme (le code CSS).

    Les attributs id et class

    En utilisant des noms de balise (par exemple p, h1, h2, etc.) dans les déclarations CSS, tous les contenus de ces balises sont mis en forme de la même façon (par exemple tous les textes des éléments <p> apparaissent en bleu foncé maintenant). Si on veut qu'une propriété particulière ne s'applique qu'à un élément précis, on peut bien sûr placer un attribut de style dans sa balise, mais ce n'est pas recommandé. La bonne approche est d'utiliser les attributs spéciaux :  id et class.

    L'attribut « id » permet d'identifier un élément de façon unique, il ne peut être utilisé que pour un seul élément de la page. L'attribut « class » permet d'identifier plusieurs éléments similaires.

    Par exemple pour mettre en forme le premier paragraphe <p> différemment des autres, il faut identifier sa balise précisément avec un nom unique, par exemple « intro ». On utilise l'attribut id="intro".

  11. Modifier le fichier « systemes-exploitation.html » an joutant cet attribut au premier paragraphe :
    12. <p id="intro">Un système d'exploitation (<i>Operating System ...</p>

    13. #intro {

    14. Les deux règles déclarées pour les sélecteurs p et #intro s'appliquent. La propriété color:darkblue est déclarée dans le sélecteur p et n'est pas remplacée par #intro, elle s'applique donc aussi à ce paragraphe.

  14. Si on veut mettre en forme plusieurs éléments ensembles, par exemple pour mettre en forme les systèmes d'exploitation de smartphone, il faut les identifier avec un attribut class car il y en a plusieurs concernés.

    Noter la différence, on utilise le caractère « # » au début d'une déclaration d' id et « . » au début d'une déclaration de class.

    Couleurs

    On peut obtenir des couleur en utilisant leur format de couleur RGB pour « Red, Green, Blue » (Rouge, Vert, Bleu en français). On indique un niveau compris entre 0 et 255 pour chaque couleur. En les mélangeant dans des proportions différentes, on peut obtenir toutes les couleurs.

  16. Modifier le fichier « styles.css » en ajoutant un sélecteur body pour modifier la couleur de fond de page :

  21. Ajouter au sélecteur body les propriétés suivantes, l'une après l'autre, et compléter le tableau.

    22. Propriété ajoutée dans le sélecteur body

    Effet sur l'élément <body> dans la page web

    width: 80%;

    border: solid black;

    padding: 50px;

    margin: auto ;

    Avec CSS, tout élément HTML peut être vu comme un empilement de boites rectangulaires :

    • La première boite, au centre, est composée du contenu de l'élément en lui-même;
    • La deuxième boite (« padding ») comprend des marges internes entre l'élément et sa bordure;
    • La troisième boîte (« border ») comprend les bordures de l'élément;
    • La quatrième boite (« margin ») comprend marges externes autour des bordures.

    Des propriétés pour toutes ces boites permettent de mettre en forme les éléments et la page web.
